# background-jobs

## Usage

### server

```sh
cd background-jobs
cargo run
# starting HTTP server at http://localhost:8080
```

### Available Routes

- [GET /cache](http://localhost:8080/cache)
- [POST /cache](http://localhost:8080/cache)
- [POST /email](http://localhost:8080/email)
- [POST /email-spam](http://localhost:8080/email-spam)
